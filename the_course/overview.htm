Overview: MarkLogic Fundamentals

Length: 1 day

In this 1-day hands-on course, you learn the basics of MarkLogic, including what it is and what it is used for, how to install it, how to create a database, how to model data, how to insert data, how to make data available to applications, how to search data, as well as how additional technologies can be used in MarkLogic.

The recommended prerequisites for this course are experience working with databases and experience working with JavaScript (or any other programming language).

 

Unit 1: Understanding MarkLogic
Unit 2: Installing MarkLogic
Unit 3: Creating a Database
Unit 4: Modeling Data
Unit 5: Bringing In Data
Unit 6: Serving Data
Unit 7: Searching Data
Unit 8: Learning About More Capabilities

 

For live training online, you will want to have two computers (or one computer with two monitors) so that you can see what the instructor is doing on one monitor, while you are working on the second monitor.

For the self-paced version of this course, you will want to do the setup steps in the self-paced course videos to ensure that your computer will be able to run the course virtual machine (downloadable from MarkLogic University) locally.

This course was written using a MarkLogic University virtual machine (for running on a Windows, Mac, or Linux computer), using CentOS in VirtualBox.

This course could also potentially be done without the course virtual machine, directly on a Linux computer (for setup, see See the Steps That Were Done), or on a Mac or on Windows (with adjustments as needed, such as file paths, and MarkLogic installation steps). For a list of platforms supported by MarkLogic, please see the MarkLogic Installation Guide.
