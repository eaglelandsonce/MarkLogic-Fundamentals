Exercise: Initialize the Instance

In this walkthrough exercise, you initialize the MarkLogic instance in your CentOS virtual machine.

 

  1. In your virtual machine, open a browser window (Applications - Internet - Firefox).
  2. Enter http://localhost:8001.
  3. Notice the message about setting up the initial databases and application servers:

  4. Click the ok button.
  5. Wait for the server to restart after it has set up the initial databases and application servers.
  6. In the Join a Cluster screen, click the skip button.
  7. In the Security Setup screen, enter admin for Admin.
  8. Enter admin for both Admin Password and Confirm Admin Password.
  9. Leave Realm with the default value of public.
  10. Enter admin for both Wallet password and Confirm Wallet password.
  11. Click the ok button.
  12. Wait for MarkLogic to create the admin user.
  13. Once the setup has completed, see that an Authentication Required window appears:

  14. Log in with a User Name of admin and a Password of admin, to log in to MarkLogic as the user that you just set up.
  15. See that you are now viewing the System Summary screen of MarkLogic's administrative interface:

  16. Realize that the Admin Interface can be used to do a wide variety of administration tasks, such as create databases, create app servers, view log files, and set up security roles.

View the license currently installed in this MarkLogic host:

  1. In the left navigation, click on Hosts.
  2. Click on the name of the host (e.g. centos7-vm).
  3. Select the Status tab.
  4. Click the license key button.
  5. See that this is where you would enter your license key information:

  6. Click the browser's back button, to return to the previous screen.
  7. If desired, continue to explore the Admin Interface, selecting items in the left navigation, noticing that the tabs on the right update, based on what you have selected.
  8. Click Configure, to return to the summary page for the Admin Interface:
