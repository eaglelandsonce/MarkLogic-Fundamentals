Lab: Create a Database

In this lab exercise, you either use the steps below (normal steps), or the alternate steps (less detailed, so more challenging), to create a new database named playground, with a forest attached to it, and a REST API instance port of 8050 for communicating with it.

 

  1. On the desktop, in the mls-fundamentals/unit03 folder, open the playground-config.json file in a text editor to see a REST API instance configuration file for creating a database named playground:


  2. In a command prompt window, enter the following text (can copy from unit03/laba.txt file):

    curl --anyauth --user admin:admin -X POST -d@"Desktop/mls-fundamentals/unit03/playground-config.json" -i -H "Content-type: application/json" http://localhost:8002/v1/rest-apis
  3. Notice the reference to the playground-config.json file.
  4. Press Enter, to generate a REST API instance for the playground database.
  5. Wait a few seconds, to see that the command is successful:


  6. In the Admin Interface, click the Configure link on the left, to see that the database, forest, and port all now exist:


  7. In the Databases column, click playground, and then select the Status tab.
  8. See that the number of forests is 1 and that the playground-1 forest is listed:
