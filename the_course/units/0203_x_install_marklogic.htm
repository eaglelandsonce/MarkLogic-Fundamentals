Exercise: Install MarkLogic

In this walkthrough exercise, you install MarkLogic on your virtual machine.

 

  1. In a browser, navigate to the Installation Guide for All Platforms at docs.marklogic.com/guide/installation.
  2. Click the Procedures link.
  3. Scroll down to the table that lists the operating systems and their corresponding steps.
  4. Scan the list, noticing that there are steps for installing on Window x64, Red Hat Linux x64, CentOS Linux, and Mac OS X.
  5. Note the CentOS Linux steps:

  6. In your virtual machine, open a command prompt window (Applications - System Tools - Terminal).
  7. To become root, enter su.
  8. For password, enter centos.
  9. Enter cd /home/cent/Downloads.
  10. Enter ls.
  11. See the MarkLogic-10.0-3.x86_64.rpm file listed:

  12. Enter yum install MarkLogic-10.0-3.x86_64.rpm.
  13. Enter y when prompted.
  14. Wait for the installation to finish:

Configure MarkLogic to run as cent:

  1. Still logged in as root, enter vim /etc/marklogic.conf.
  2. Press the i key on your keyboard, to be able to edit the document.
  3. Enter MARKLOGIC_USER=cent.
  4. Press the Esc key on your keyboard.
  5. Type :wq and then press the Enter key, to exit.
  6. Leave the command prompt window open, for the next exercise.
