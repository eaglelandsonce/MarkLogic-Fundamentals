Exercise: Access Upcoming Classes

In this walkthrough exercise, you view upcoming MarkLogic University classes.

 

  1. In a browser, navigate to mlu.marklogic.com.
  2. Scroll down through the page, noting the learning tracks, link to upcoming classes, self-paced training, on-demand tutorials, learning assessments, and certification.
  3. Scroll back up, to click the Courses link:

  4. Select a course of interest to you (e.g. MarkLogic Administration).
  5. Skim through the description and outline.
  6. Click the SELECT YOUR DAY AND TIME TO REGISTER button, to see the date(s) and time zone for when the course is being offered.
  7. If desired, register for the course.
  8. In a new browser window, navigate to mlu.marklogic.com/selfpaced.
  9. Notice that some courses are available in a self-paced format:


  10. If desired, click the Self-Paced link for a course of interest (e.g. MarkLogic Data Integration).
  11. See that the course is available now.
  12. If desired, start the course.
