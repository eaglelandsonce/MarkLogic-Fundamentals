Exercise: View the Documentation

In this walkthrough exercise, you access the MarkLogic documentation, for learning more about how MarkLogic works.

 

  1. In a browser window, navigate to docs.marklogic.com.
  2. See that you are on the landing page for the MarkLogic documentation:


  3. Scroll down to see the many guides under the various sections (e.g. Getting Started Guides, Developer's Guides, Administrator's Guides, Other Reference Guides).
  4. In the Developer's Guides section, notice the REST Application Developer's Guide, the Java Application Developer's Guide, the Node.js Application Developer's Guide, and the Application Developer's Guide.
  5. In the Administrator's Guides section, notice the Administrator's Guide.
  6. In the Other Reference Guides section, notice the Scalability, Availability, and Failover Guide and the Security Guide.
  7. If desired, select the PDF icon for the Security Guide:


  8. Skim the table of contents.
  9. If desired, click a topic of interest (e.g. 2. Role-Based Security Model (Authorization)).
  10. Skim through the topic.
  11. Continue perusing the guides, as desired.
