Exercise: Search Triples

In this walkthrough exercise, you query the triples that you loaded into the star-wars database in Unit 5.

 

  1. In Query Console, with the star-wars database selected, click the Explore button.
  2. Locate the /triplestore/ document:

  3. Click the document name, to view the contents:


  4. Know that, instead of viewing a /triplestore/ document, it is more common to use SPARQL to query triples.
  5. Import the Search Triples.xml workspace from the unit07 folder.
  6. On the first tab, Han Solo, see that you are searching your triples for information about Han Solo:

    ﻿SELECT ?p ?o
    WHERE {<http://marklogic.com/mlu/star-wars/resource/Han_Solo> ?p ?o}
  7. With the star-wars database selected, and SPARQL Query selected as the Query Type, click Run.
  8. See that the names of actors who played Han Solo are returned:


  9. Select the More tab:

    ﻿SELECT ?o ?p2 ?o2
    WHERE {<http://marklogic.com/mlu/star-wars/resource/Han_Solo> <http://marklogic.com/mlu/star-wars/ontology/playedBy> ?o . ?o ?p2 ?o2}
  10. Realize that this query is to find any data in the triples about the actors that played Han Solo.
  11. Click Run.
  12. See that data about the actors that played Han Solo is returned:


  13. Select the Just Alma Mater tab:

    ﻿SELECT ?o ?o2
    WHERE {<http://marklogic.com/mlu/star-wars/resource/Han_Solo> <http://marklogic.com/mlu/star-wars/ontology/playedBy> ?o . ?o <http://marklogic.com/mlu/ontology/almaMater> ?o2}
  14. Click Run.
  15. See that just the alma maters for actors that played Han Solo are returned:

  16. Imagine querying to find out what other famous actors have the same alma mater as actors who played Han Solo.
  17. In a browser window, navigate to www.marklogic.com/learn/courses/using-marklogic-semantics.
  18. Scroll down to skim the table of contents:

  19. If desired, sign up for the self-paced (or instructor-led) version of the course.
