Exercise: See How MarkLogic Is Being Used

In this walkthrough exercise, you access examples of MarkLogic in use.


  1. In a browser, navigate to www.marklogic.com/customers.
  2. Scroll down to peruse some of the customer case studies:


  3. Select a case study that looks interesting to you (e.g. OECD).
  4. Skim through the page, noting why the organization chose MarkLogic.
  5. Click the browser's Back button, to return to the list of featured case studies.
  6. Select another one (e.g. NBC or ABN AMRO).
  7. Skim through the page, noting why the organization chose MarkLogic.
  8. Click the browser's Back button, to return to the list of featured case studies, and continue perusing.
  9. In a browser window, navigate to www.yell.com.
  10. Realize that this application uses MarkLogic for storing its data.
  11. In the search field, enter thai.
  12. Notice the auto-complete feature, retrieving values from the database:


  13. Select thai restaurants.
  14. Enter London for the city and click the Search button.
  15. See results listed, with a View Map button.
  16. Click the View Map button:


  17. See the listings with the map:


  18. Realize that MarkLogic's geospatial functionality is being used for the location of the restaurants.
  19. In a browser, navigate to www.marklogic.com/blog/bbc-olympics-2012.
  20. Notice that this blog post contains a video demonstration of the 2012 BBC Olympics site, which uses MarkLogic to store and serve content:


  21. If desired, watch the first couple minutes of the video.
  22. In a browser, navigate to ldip.fairfaxcounty.gov.
  23. In the search fields, enter "swimming pools" for All text, and 01/01/2019 for After Date, to access all of the land development records that reference a swimming pool, from Jan 1, 2019 onwards:


  24. Click the Submit button to see the results:


  25. Notice that, because of MarkLogic's stemming feature, records with "swim pool" were also found.
  26. If desired, navigate to www.marklogic.com/solutions/industry.
  27. See MarkLogic solutions per industry listed:


  28. If desired, select an industry of interest (e.g. Insurance).
  29. Continue perusing, as desired.
