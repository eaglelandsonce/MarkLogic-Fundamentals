Exercise: Use Global Search

In this walkthrough exercise, you use global search to access a variety of MarkLogic assets.

 

  1. In a browser, navigate to globalsearch.marklogic.com.
  2. Make the browser window wide enough to be able to see the categories on the left.
  3. Notice the sources:

  4. Notice the categories:

  5. Select a category of interest (e.g. Blogs - Technical).
  6. See that the assets are narrowed down to that category:

  7. Deselect the category checkbox, to bring back all of the assets.
  8. At the top of the page, in the Search field, type white paper.
  9. See that the suggestions contain the phrase white paper:

  10. If desired, select one of the suggestions (e.g. Inside MarkLogic Server White Paper) and click the Search icon.
  11. If desired, view the resource and peruse the content.
  12. Continue exploring the Global Search interface, as desired.
