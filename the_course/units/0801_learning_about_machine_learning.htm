Learning About Machine Learning

With the release of MarkLogic 10.0-2, and further enhanced in 10.0-3, machine learning models can be run within MarkLogic, avoiding the need to export data to a third party system, run models, and then import the results back into MarkLogic. Instead, models can be stored and run directly in MarkLogic.

ONNX Runtime

As of MarkLogic 10.0-3, the ONNX (Open Neural Network Exchange) Runtime is embedded in MarkLogic.

ONNX is a open format for machine learning models, providing an open standard for moving models between the various framework technologies.


With the ONNX Runtime embedded in MarkLogic 10.0-3, different development teams in an organization may each use different machine learning frameworks to create their models. By each exporting their model into the ONNX format, their model can be used in MarkLogic.

MarkLogic 10.0-2 embedded the CNTK (Microsoft Cognitive Toolkit) libraries, which are now deprecated, in favor of ONNX Runtime.
