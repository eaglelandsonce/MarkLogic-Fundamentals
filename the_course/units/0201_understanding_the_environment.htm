Understanding the Environment

MarkLogic can run in a variety of environments, including bare metal (i.e. physical computers), virtual machines, containers, cloud platforms, or the MarkLogic Data Hub Service.

Data Hub Service

MarkLogic Data Hub Service is a MarkLogic cloud service that uses the MarkLogic Data Hub platform.

Detail on the MarkLogic Data Hub Service is available at developer.marklogic.com/concept/data-hub-service.

Cloud Platforms

MarkLogic can be run on cloud platforms. Two cloud platforms of special note, which MarkLogic supports with an official partnership as well as documentation, are AWS (Amazon Web Services) and Microsoft Azure.

Detail on running MarkLogic on AWS is at docs.marklogic.com/guide/ec2. Detail on running MarkLogic on Azure is at docs.marklogic.com/guide/azure.

Containers

MarkLogic can be run in containers, such as Docker. In addition, MarkLogic supports Red Hat UBI (Universal Base Image).

Detail on running MarkLogic on Docker is available at www.marklogic.com/product/marklogic-database-overview/database-features/containers. Detail on Red Hat UBI is at www.marklogic.com/news/marklogic-developers-achieve-boosted-speed-and-agility-with-the-red-hat-universal-base-image-for-containers.

Virtual Machines

MarkLogic can be run in virtual machine environments, such as VMware or VirtualBox.

For best practices on using MarkLogic in VMware, see help.marklogic.com/Knowledgebase/Article/View/168/0/vmware-best-practices-guide.

Bare Metal

MarkLogic can be installed directly on any computer that meets the MarkLogic system requirements.

System Requirements

MarkLogic can be installed on the following 64-bit operating systems:

  • Red Hat Enterprise Linux 7
  • CentOS 7
  • Mac OS X 10.14* or later
  • Windows Server 2012, 2016, or 2019
  • Windows 10*
  • Amazon Linux
  • Azure Windows Server 2012, Red Hat Enterprise Linux 7, or CentOS 7
  • Docker

*Mac OS and Windows 10 are for development only

For RAM, MarkLogic requires a minimum of 4 GB of system memory, with 8 GB or more recommended.

For disk space, MarkLogic requires at least 1.5 times the size of the loaded source content, to allow for merging.

Detail on the system requirements is available at docs.marklogic.com/guide/installation/intro#chapter.
