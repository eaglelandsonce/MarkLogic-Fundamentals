Creating a Forest

A forest can be created through a GUI or programmatically.

In the Admin Interface, the Create tab in the Forests section can be used to create a forest.

When creating a forest, the only mandatory field is forest name, to give the forest a name.

The data directory field is for specifying a location for the forest on the host computer. If this field is left blank, the forest will be saved in the default directory. The default data directory on Linux is /var/opt/MarkLogic/Forests, the default data directory on Windows is C:\Program Files\MarkLogic\Data\Forests, and the default data directory on the Mac is ~/Library/Application\ Support/MarkLogic/Data.

Detail on creating and working with forests is available in the Forests section of the Administrator's Guide, available at docs.marklogic.com/guide/admin/forests.
